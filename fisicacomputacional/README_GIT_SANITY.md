# PDG · GIT PUSH REMEDIATION CAPSULE — fisicacomputacional

Canonical remediation capsule for resolving HTTP push failures (2026-04-16).

Summary
-------
- Symptoms: HTTP 408 / 500 during `git push`, `send-pack` disconnects, remote rejects due to pack > 2 GiB.
- Root cause: excessive repository payload (large media/binaries) causing transport failures.

Quick actions (copy-paste)
---------------------------
1) Audit

```bash
git fetch origin
git status -sb
git log --oneline origin/main..main
git log --oneline main..origin/main
git diff --cached --stat
git count-objects -vH

git rev-list --objects --all | \
  git cat-file --batch-check='%(objecttype) %(objectname) %(objectsize) %(rest)' | \
  sort -k3 -n | tail -30
```

2) Ignore generated / heavy files (example additions to `.gitignore`)

Add common build artifacts and optional heavy files (videos, archives) to `.gitignore`.

3) If many large tracked files exist — migrate to Git LFS

```bash
git lfs install
git lfs track "*.mp4" "*.mov" "*.pdf" "*.pptx" "*.zip" "*.tar" "*.7z"
git add .gitattributes
git commit -m "Track large media and archives with Git LFS"

# Migrate history (rewrites history on main) — careful, this requires force-push
git lfs migrate import --include="*.mp4,*.mov,*.mkv,*.avi,*.pptx,*.pdf,*.zip,*.tar,*.gz,*.rar,*.7z" --include-ref=refs/heads/main
git reflog expire --expire=now --all && git gc --prune=now --aggressive
git lfs push --all origin
git push origin main --force
```

4) Transport tweaks (if you cannot use SSH immediately)

```bash
git config --global http.version HTTP/1.1
git config --global core.compression 0
git config http.postBuffer 524288000
git push origin main
```

5) Switch to SSH (recommended for reliability)

```bash
git remote set-url origin git@github.com:jbermejovega/fisicacomputacional.git
git push origin main
```

Notes and safety
----------------
- Migrating to Git LFS rewrites history and requires a coordinated force-push; inform collaborators.
- If you want me to perform the migration now, confirm and ensure you have backups of any uncommitted work.
